# เชื่อมต้นทุนเครื่องดื่มกับ Google Sheet (สองทาง: อ่าน + เขียนกลับ)

ทำครั้งเดียว แล้วหลังจากนั้นแก้ราคา/สูตรได้จาก **Google Sheet หรือจากหน้า dashboard** — ซิงก์หากันอัตโนมัติ ไม่ต้องแตะ Excel / git อีก

---

## ขั้นที่ 1 — สร้าง Google Sheet
1. เปิด https://sheets.google.com → สร้างสเปรดชีตใหม่ ตั้งชื่ออะไรก็ได้ (เช่น "ต้นทุนเครื่องดื่ม")
2. ไม่ต้องสร้างแท็บเอง — สคริปต์จะสร้าง `วัตถุดิบ / เมนู / สูตร / ตั้งค่า` ให้ตอนกด "ดันข้อมูลขึ้น Sheet" ครั้งแรก

## ขั้นที่ 2 — ติดตั้งสคริปต์
1. ใน Sheet: เมนู **ส่วนขยาย (Extensions) → Apps Script**
2. ลบโค้ดเดิมทิ้ง แล้ววางเนื้อหาทั้งหมดจากไฟล์ **`Code.gs`** (ในโฟลเดอร์นี้)
3. กดบันทึก (รูปแผ่นดิสก์ / Ctrl+S)

## ขั้นที่ 3 — Deploy เป็น Web App
1. มุมขวาบน: **Deploy → New deployment**
2. ไอคอนเฟือง ⚙️ → เลือก **Web app**
3. ตั้งค่า:
   - **Execute as:** Me (อีเมลคุณ)
   - **Who has access:** **Anyone**  ← สำคัญ (ให้ dashboard เรียกได้)
4. กด **Deploy** → อนุญาตสิทธิ์ (Authorize) ตามขั้นตอน (เลือกบัญชี → Advanced → ไปต่อ → Allow)
5. คัดลอก **Web app URL** (ลงท้ายด้วย `/exec`)

> แก้สคริปต์ภายหลัง: ต้อง **Deploy → Manage deployments → ✏️ → Version: New version → Deploy** ทุกครั้ง (ไม่งั้น URL เดิมยังรันโค้ดเก่า)

## ขั้นที่ 4 — เชื่อมในหน้า dashboard
1. เปิดหน้า **ต้นทุนเครื่องดื่ม** → กด **✏️ แก้ไขเมนู/สูตร** → **🔗 เชื่อม Sheet**
2. วาง Web app URL → กด **ดันข้อมูลปัจจุบันขึ้น Sheet** (เอา 22 เมนูตั้งต้นขึ้นไปก่อน) → กด **บันทึก URL**
3. เสร็จ! ลองแก้ราคาในเมนูดู — มันจะเขียนกลับ Sheet อัตโนมัติ และถ้าแก้ใน Sheet แล้วกด **🔄 ซิงก์** dashboard จะดึงค่าล่าสุด

## โครงสร้างแท็บ (ถ้าจะกรอกใน Sheet เอง)
| แท็บ | คอลัมน์ |
|---|---|
| **วัตถุดิบ** | ชื่อ · ราคา · ปริมาณ · หน่วย |
| **เมนู** | ชื่อ · หมวด (signature/coffee/tea/milk/soda) · หน้าร้าน · Lineman · Shoppee · **Grab** · ต้นทุนตั้งต้น · รูปภาพ (URL) |
| **สูตร** | เมนู (ชื่อตรงกับแท็บเมนู) · วัตถุดิบ (ชื่อตรงกับคลัง) · ปริมาณ |
| **ตั้งค่า** | คีย์ · ค่า (overhead, margin_factor, fixed_cost_monthly, days_per_month, gp_lineman, vat_lineman, gp_shoppee, vat_shoppee, **gp_grab**, **vat_grab**) |

> ⚠️ **หลังอัปเดต Code.gs ครั้งนี้** (เพิ่มคอลัมน์ Grab และ gp_grab/vat_grab) ต้อง Deploy → Manage deployments → ✏️ → **Version: New version → Deploy** 1 ครั้ง — มิฉะนั้น URL เดิมยังรันสคริปต์เวอร์ชันเก่าที่ไม่รู้จักคอลัมน์ Grab

ต้นทุน/แก้ว = Σ(ราคา÷ปริมาณ × ปริมาณที่ใช้) × (1 + overhead) · เดลิเวอรีหัก GP + VAT ของช่องทางนั้น

## ความปลอดภัย
- Web app URL = ใครมีลิงก์ "อ่าน+เขียน" Sheet ได้ → **อย่าแชร์ URL นี้ในที่สาธารณะ**
- dashboard เก็บ URL ไว้ใน localStorage ของเบราว์เซอร์คุณเท่านั้น (ไม่ฝังในโค้ดเว็บที่ push ขึ้น GitHub)
- ถ้าหลุด: ไป Apps Script → Manage deployments → Archive แล้ว Deploy ใหม่ (ได้ URL ใหม่)
