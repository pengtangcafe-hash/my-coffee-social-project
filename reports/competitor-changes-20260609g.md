## ข้อสรุปเชิงกลยุทธ์

### Event Venue = ตลาดที่ Pengtang ยังไม่ถึง
The Mountain เปิดรับ B2B (สัมมนา, งานแต่ง, ทัวร์กลุ่ม) = revenue stream ที่ local café ทั่วไปไม่มี

### Local Ingredient Branding ที่น่าจับตา
- **ไวน์หมากเม่า แบรนด์บ้านเอง** — เหมือนคอนเซปต์ Hook Cafe (น้ำหมากเม่าไนโตร) แต่เป็น wine format
- **กาแฟน้ำช่อดอกมะพร้าว** — Thai-local signature ชัดเจน

### โอกาสของ Pengtang
1. Wongnai 1 review เท่านั้น — Pengtang สร้าง review base ก่อนยังได้
2. ปิดวันพุธ — Pengtang เปิดทุกวันจะ capture วันพุธทั้งหมด
3. The Mountain เน้น event + full meal → Pengtang focus daily coffee ไม่ overlap segment หลัก
